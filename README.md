# faz_marido
# Faz Marido  Plataforma web para conectar profissionais de serviços domésticos com clientes. Sobre o Projeto  Desenvolvi este projeto para solucionar a dificuldade de encontrar profissionais confiáveis para pequenos reparos domésticos na região de Três Rios. O "Faz Marido" conecta prestadores de serviços com clientes.
